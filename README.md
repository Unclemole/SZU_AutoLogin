# 深圳大学办公区自动化程序
1、请确保软件与chromedrive.exe在同一目录，并确保chromedrive版本号与你的chrome版本一致，请前往https://googlechromelabs.github.io/chrome-for-testing/下载对应版本，或使用一键下载器下载（确保你的网络能访问Google）  
2、本软件仅能在SZU办公区使用，如果使用代理软件请绕行https://net.szu.edu.cn/（clash在订阅处右键你的配置文件，点击编辑规则，将net.szu.edu.cn设置为DIRECT加入规则）  
3、cred.dat为你的加密密码信息，请不要删除  
4、如果勾选“保存密码”与“开机启动”，软件在下次开机启动时会自动进入登录/挂机模式并自动隐藏进托盘
5、软件的清除按键在记住密码处于未勾选状态时会清除当前账户，托盘处的清除账户为清除所有账户，请谨慎使用
6、本软件仅供个人使用，如若违反相关规定或侵害他人权利请自行删除
